# E-Citizen Health Cabin - Business Requirements Document

## 1. Tổng quan dự án

**Tên dự án:** E-Citizen Health Cabin (Cabin Y tế Thông minh)

**Mục tiêu:** Xây dựng giao diện màn hình cho cabin y tế phục vụ người dân, tích hợp các dịch vụ y tế công cộng.

---

## 2. Phạm vi

### 2.1 Chức năng chính
- [ ] Màn hình chào mừng / Standby
- [ ] Đo sinh hiệu cơ bản
- [ ] Tư vấn sức khỏe tự động
- [ ] Kết nối bác sĩ từ xa (Telemedicine)
- [ ] In/gửi kết quả khám

### 2.2 Yêu cầu kỹ thuật
- Frontend: Next.js + TypeScript + Tailwind CSS
- Backend: .NET 8 Clean Architecture
- Giao diện: Màn hình cảm ứng, responsive cho cabin

---

## 3. Stakeholders

| Vai trò | Trách nhiệm |
|---------|-------------|
| Product Owner | Định hướng sản phẩm |
| UI/UX Designer | Thiết kế giao diện cabin |
| Frontend Dev | Xây dựng giao diện |
| Backend Dev | Xây dựng API |
